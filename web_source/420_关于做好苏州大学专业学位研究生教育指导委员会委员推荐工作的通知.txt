				  <div class="entry">
					<div class="read"><div class='wp_articlecontent'><p style="font-size:16px;">各联盟企业：</p><p style="font-size:16px;">&nbsp; &nbsp; 学校公布《关于做好苏州大学专业学位研究生教育指导委员会委员推荐工作的通知》，企业也可以申报担任委员，有意者请下载并填写三张表格于4月9日下午16：00点前回发给本人邮箱wangdong@suda.edu.cn.</p><p style="font-size:16px;">&nbsp;</p><p style="text-align:right;font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 苏州大学IT校企合作联盟</p><p style="text-align:right;font-size:16px;">&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 2019年4月1日</p><p style="font-size:16px;"><br /></p><p style="font-size:16px;">研字【2019】32号<br />&nbsp; &nbsp; 各相关研究生培养单位：<br />根据《苏州大学关于深化硕士专业学位研究生教育综合改革的指导意见》（苏大研【2018】70号）文件，我校拟组建专业学位研究生教育指导委员会（以下简称“教指委”），现将教指委委员的有关推荐工作通知如下：<br />一、&nbsp; 类别设置<br />学校本次按照国家专业学位类别设置，结合本校专业学位授权点实际，拟设置24个教指委，分别是金融、税务、国际商务、教育、应用心理、体育、社会工作、法律、会计、翻译、新闻与传播、汉语国际教育、风景园林、艺术、出版、应用统计、工商管理、公共管理、农业、药学、临床医学、公共卫生、护理和工程。<br />二、&nbsp; 人员组成<br />1.&nbsp; 委员人选<br />教指委委员主要来自研究生培养单位负责人、专业学位点负责人，从事硕士专业学位研究生教育的副高级职称以上的专家，相关职能部门负责人以及校外相关行业专家。每个教指委委员人数原则上为7-9人，其中校外行业专家不低于1人。每个教指委设主任委员1名，副主任委员1-2名，秘书处设在主任委员所在研究生培养单位，设秘书一名。<br />2.&nbsp; 聘期管理<br />各教指委由学校聘任，每届任期4年，连续聘任一般不超过两届。聘任期间不宜继续担任委员的，由学校委托所在教指委进行调整。任期届满后由学校组织开展换届工作。<br />三、&nbsp; 教指委委员推荐条件<br />1. 热心从事专业学位研究生教育事业。<br />2. 在本专业领域具有较强的专业背景，原则上应具有副高级职称及以上。<br />3. 身体健康，在学校规定的退休年龄前能够完成一届教指委工作。<br />4. 思想素质好，作风正派，办事公正。<br />四、&nbsp; 工作流程<br />1. 研究生培养单位推荐<br />（1）专业学位类别设在一个研究生培养单位的，直接由所在研究生培养单位推荐，并由所在研究生培养单位研究生教育负责人召集推荐人员召开会议，初步选举产生拟推荐的1名主任委员和1-2名副主任委员后与推荐委员名单一并报送研究生院。<br />（2）一个专业学位类别同时设在不同研究生培养单位的，由相关研究生培养单位推荐委员共同组成本教指委。<br />金融类别由东吴商学院作为联系人汇总推荐委员名单，艺术类别由艺术学院作为联系人汇总推荐委员名单，并由东吴商学院和艺术学院研究生教育负责人各自召集推荐人员，初步选举产生拟推荐的1名主任委员和1-2名副主任委员后与推荐委员名单一并报送研究生院。<br />教育、临床医学和工程三个类别由相关研究生培养单位推荐，直接将推荐委员名单报送研究生院，每个培养单位推荐1-2名委员。<br />2. 学校审核、发文聘任<br />研究生院汇总各研究生培养单位推荐人员后，报送学校审核，学校审核通过后发文聘任。<br />五、&nbsp; 推荐人员报送时间安排<br />请相关研究生培养单位于2019年4月10日前根据推荐、报送安排表（附件1）完成教指委委员推荐工作，并填写《苏州大学专业学位研究生教育指导委员会推荐委员人选表》（附件2，其中教育、临床医学和工程三个类别只填报推荐委员）以及《苏州大学专业学位研究生教育指导委员会推荐委员个人简况表》（附件3）。以上表格（附件2和附件3）纸质版请于4月10日前报研究生院培养办公室（地址：蕴秀楼211室），同时将电子版打包以“XX专业学位研究生教育指导委员会推荐委员材料”命名发送至liuy@suda.edu.cn。<br />六、&nbsp; 其它事项<br />欢迎从事专业学位研究生教育的专家、学者自荐，成为教指委的一员，为学校专业学位研究生教育事业发展和人才培养献计献策。未尽事宜，请与研究生院联系:<br />联系人：刘老师<br />联系电话：65226986<br />邮箱：liuy@suda.edu.cn。<br />特此通知。<br /></p><div><br />附件1 苏州大学专业学位研究生教育指导委员会委员推荐安排表.docx<img src="/_ueditor/themes/default/images/icon_doc.gif" /><a href="/_upload/article/files/fc/9a/305902d14f008fc492e0ec2fd2f7/94d76a9e-0b55-4701-a9b7-f0c3374a3aea.docx" sudyfile-attr="{'title':'附件1 苏州大学专业学位研究生教育指导委员会委员推荐安排表.docx'}">附件1 苏州大学专业学位研究生教育指导委员会委员推荐安排表.docx</a><br /></div><div>附件2 苏州大学专业学位研究生教育指导委员会推荐委员人选表.xlsx<img src="/_ueditor/themes/default/images/icon_xls.gif" /><a href="/_upload/article/files/fc/9a/305902d14f008fc492e0ec2fd2f7/55843d89-518b-4b74-b8f6-b5dbd27694b5.xlsx" sudyfile-attr="{'title':'附件2 苏州大学专业学位研究生教育指导委员会推荐委员人选表.xlsx'}">附件2 苏州大学专业学位研究生教育指导委员会推荐委员人选表.xlsx</a><br /></div><div>附件3 苏州大学专业学位研究生教育指导委员会推荐委员个人简况表.docx<img src="/_ueditor/themes/default/images/icon_doc.gif" /><a href="/_upload/article/files/fc/9a/305902d14f008fc492e0ec2fd2f7/882ee48d-be96-4e97-8762-770b66637151.docx" sudyfile-attr="{'title':'附件3 苏州大学专业学位研究生教育指导委员会推荐委员个人简况表.docx'}">附件3 苏州大学专业学位研究生教育指导委员会推荐委员个人简况表.docx</a><br /></div><div style="text-align:right;">研究生院<br />2019-03-19 08:31:14</div><p><br /></p></div></div>
