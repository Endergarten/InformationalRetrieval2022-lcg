<p style="padding:0px;border:0px;color:#666666;font-family:arial, 宋体, 微软雅黑;font-size:14px;line-height:32px;background-color:#ffffff;margin-top:0px;margin-bottom:0px;"><strong style="margin:0px;padding:0px;border:0px;"><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#000000;">1、计算机实验教学中心</span></strong><br style="margin:0px;padding:0px;border:0px;" /><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#000000;">&nbsp; &nbsp; &nbsp;“计算机与信息技术实验教学中心”现为国家级实验教学示范中心。中心主要为学校和学院师生的教学服务，中心下设计算机公共基础教学实验室、计算机专业教学实验室、学生开放实验室和实训中心等。<br style="margin:0px;padding:0px;border:0px;" />&nbsp; &nbsp; <strong>&nbsp; &nbsp;专业教学实验室</strong><br style="margin:0px;padding:0px;border:0px;" />&nbsp; &nbsp; &nbsp; &nbsp;主要为计算机科学与技术学院各专业的教学与科研服务。目前建有计算机硬件基础实验室、计算机软件基础实验室、（操作系统分析实验室、）嵌入式系统基础实验室、计算机多媒体基础实验室、计算机软件工程实验室、计算机软件测试实验室、中文信息处理实验室、计算机组网实验室、TCP/IP协议分析实验室、信息安全实验室、数据库实验室、嵌入式系统应用实验室和信息（系统）管理实验室等。<br style="margin:0px;padding:0px;border:0px;" />&nbsp; &nbsp; &nbsp; &nbsp;<strong>计算机公共基础教学实验室</strong><br style="margin:0px;padding:0px;border:0px;" />&nbsp; &nbsp; &nbsp; &nbsp;主要为全校非计算机专业学生提供计算机基本操作技术实习、基本编程能力实习和基础应用的软件与硬件实习环境等，2005年成为江苏省计算机基础教学实验示范中心建设点。<br style="margin:0px;padding:0px;border:0px;" />&nbsp; &nbsp; &nbsp; &nbsp;<strong>学生开放实验室</strong><br style="margin:0px;padding:0px;border:0px;" />&nbsp; &nbsp; &nbsp; &nbsp;学生开放实验室主要为计算机科学与技术学院学习成绩优异且有较强科研能力的学生服务。该实验室电脑由学生自备，学院为其提供场地并安排专业指导教师给予指导，学生在规定期限内完成一定的科研项目。<br style="margin:0px;padding:0px;border:0px;" />&nbsp; &nbsp; &nbsp; &nbsp; <strong>实训中心</strong><br style="margin:0px;padding:0px;border:0px;" />&nbsp; &nbsp; &nbsp; &nbsp; 实训中心根据计算机专业人才培养计划，参照企业的培养思路、模式和方法，在实训中心进行企业文化、工作作风、工程实践等方面的综合训练。实训中心与韩国又松大学组建软件外包开发中心和与国内澳洋医药组建信息技术研究中心，形成了校企合作、项目驱动的实训教学体系，为科研和人才培养工作的开展构筑了坚强的基础。<br style="margin:0px;padding:0px;border:0px;" /></span><strong style="margin:0px;padding:0px;border:0px;"><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#000000;">2、实习基地</span></strong><br style="margin:0px;padding:0px;border:0px;" /><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#000000;">&nbsp; &nbsp; &nbsp; &nbsp;学院目前已在苏州工业园区、苏州新区、昆山等地区的四十余家IT企业建立了教学、科研与实习基地。在此基础上，学院于2016年成立“苏州大学IT校企合作联盟”，目前已有联盟单位40余家，为学院人才培养提供支持。<br style="margin:0px;padding:0px;border:0px;" /></span><strong style="margin:0px;padding:0px;border:0px;"><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#000000;">3、人才培养基地<br style="margin:0px;padding:0px;border:0px;" /></span></strong><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#000000;">&nbsp; &nbsp; &nbsp; 学院为江苏省服务外包人才培养试点高校（苏教高[2011]16号），并拥有“苏州大学-方正国际软件有限公司工程实践教育中心”国家级大学生校外实践教育基地，为人才培养提供了良好的资源。</span></p><p style="padding:0px;border:0px;color:#666666;font-family:arial, 宋体, 微软雅黑;font-size:14px;line-height:32px;background-color:#ffffff;margin-top:0px;margin-bottom:0px;"><strong style="margin:0px;padding:0px;border:0px;"><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#000000;">4、图书期刊</span></strong><br style="margin:0px;padding:0px;border:0px;" /><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#7f7f7f;"><span style="font-size:16px;font-family:微软雅黑, &quot;microsoft yahei&quot;;color:#000000;">&nbsp;
