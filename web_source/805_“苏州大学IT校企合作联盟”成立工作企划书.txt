								  <h1 class="arti_title">“苏州大学IT 校企合作联盟”成立工作企划书</h1>
				  <p class="arti_metas"><span class="arti_publisher">发布者：</span><span class="arti_update">发布时间：2016-11-17</span><span class="arti_views">浏览次数：<span class="WP_VisitCount" url="/_visitcountdisplay?siteId=289&type=3&articleId=262683">205</span></span></p>
				  <div class="entry">
					<div class="read"><div class='wp_articlecontent'><p align="center"> <strong>&ldquo;苏州大学IT 校企合作联盟&rdquo;成立工作企划书</strong></p><p> &nbsp; &nbsp; &nbsp; &nbsp;为积极响应国家全新驱动发展的战略，推动苏州及周边地区ＩＴ产业的技术创新和相关企业的转型升级，充分发挥苏州大学的人才优势和苏州ＩＴ相关企业的产业集群优势，实现产学研用强强联合，进一步推动新型ＩＴ人才的培养，为苏州ＩＴ企业输送更多符合社会和企业需要的创新型人才。苏州大学计算机科学与技术学院在苏州市相关部门和苏州市计算机学会的指导下，拟联合苏州ＩＴ相关企业成立&ldquo;苏州大学ＩＴ校企合作联盟&rdquo;，以此构建面向未来的校企业合作的平台。</p><p> <strong>&nbsp; &nbsp; &nbsp; &nbsp; 一、&nbsp; </strong><strong>联盟成立的背景</strong></p><p> &nbsp; &nbsp; &nbsp; &nbsp;苏州的ＩＴ相关企业数千家，IT产业是苏州重要的产业支撑。互联网＋、大数据、云计算及企业信息管理等技术都已成为企业转型升级的重要手段和技术支撑。互联网经济必将成为引领企业转型升级的新的重要的增长极。为响应江苏省十三五人才发展规划和互联网经济发展规划建立&ldquo;一支高水平互联网企业领军人才队伍、一支高水平互联网中坚骨干人才队伍、一支互联网服务保障人才队伍及一支互联网跨界融合人才队伍&rdquo;四支队伍的要求；适应苏州ＩＴ相关企业自主创新和转型升级的需求，高校ＩＴ相关专业创新人才的培养工作已成为社会、业界和高等学校普遍关心的热点和迫切需求。同时也为高校推动人才培养供给侧改革提供了机遇和政策保障。</p><p> <strong>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;二、&nbsp; </strong><strong>联盟成立的宗旨</strong></p><p> &nbsp; &nbsp; &nbsp; 联盟将积极推进学研产相结合，密切与国家、省、市政府科技主管部门的政策关系；</p><p> &nbsp; &nbsp; &nbsp;充分利用苏州大学与苏州市政府共同推动的&ldquo;名城名校&rdquo;战略及苏州工业研究院的平台，积极推进IT领域的社会服务、科技开发与成果转化平台；</p><p> &nbsp; &nbsp; &nbsp;联盟以联合培养企业探索企业参与苏州大学计算机学科建设和人才培养的新模式；</p><p> &nbsp; &nbsp; 联盟将与相关企业共同探索校企联合攻关、联合申请各级政府的科技和产业创新的项目，推动企业技术创新；</p><p> &nbsp; &nbsp; 联盟也将在苏州市计算机学会的指导下，构建互联网经济的信息发布平台，为校－企、企-企交流和行业信息的互通提供便利。</p><p> <strong>&nbsp; &nbsp; &nbsp; &nbsp;三、联盟活动内容</strong></p><p> &nbsp; &nbsp; 1、积极落实苏州大学与苏州市政府之&ldquo;名城名校&rdquo;战略。根据区域产业发展需求以及企业科技需求，协助企业进行技术开发、技术转让、技术难题攻关，为企业技术转型、升级提供科技支撑和人才智力支持。</p><p> &nbsp; &nbsp; 2、充分发挥联盟科技成果转化的媒介作用。在科技成果方面，学院将把联盟作为科技成果中产业的媒介机构，及时向企业提供人工智能技术、大数据处理、网络信息安全等方面的最新科技成果和信息。</p><p> &nbsp; &nbsp;3、积极争取政府资源支持。在产业政策方面，联盟将在苏州大学科技产业处及苏州市计算机学会的指导下，在科技成果转化和产业化方面提供技术和政策咨询，积极争取地方科技主管部门优先推荐科研成果申报国家、省科技专项。</p><p> &nbsp; &nbsp;4、进一步加强人才培养。以国家级大学生实践教育基地为平台，建好一批院、校两级重点是实习基地；组建专业建设咨询委员会，聘请企业技术主管为&ldquo;双师型&rdquo;导师，；开展IT 论坛，邀请企业负责人来校开设导论课程。</p><p> &nbsp;&nbsp; 5、搭建有效的沟通平台，加强行业内信息交流。建立学院与企业、企业与企业，企业与政府的定期沟通机制，邀请政府主管部门领导和学科领域内专家来联盟讲座，举办企业家论坛和沙龙，及时提供行业内最新的政策信息和和科研信息。</p><p> &nbsp; 6、协助企业做好人员培训和企业文化建设。根据IT信息行业的发展需求，以和国家级实验教学示范中心为平台，为企业的技术革新、项目开发、员工培训、文化建设产业发展等方面工作提供场所和服务。</p><p> <strong>　 &nbsp; &nbsp; &nbsp; &nbsp; 四、联盟运行机构</strong></p><p style="margin-left: 31.5pt"> 联盟最高权力机构：会员代表大会；</p><p style="margin-left: 31.5pt"> 联盟执行机构：理事会</p><p style="margin-left: 31.5pt"> 联盟运行机构：秘书处；</p><p style="margin-left: 31.5pt"> 联盟设立理事长单位和副理事长单位；</p><p style="margin-left: 31.5pt"> 联盟设立理事长1 人、副理事长和荣誉理事长若干名，并设立秘书长1名和副秘书长若干名</p><p style="margin-left: 31.5pt"> <strong>五、 联盟运行机制</strong></p><p> &nbsp; &nbsp; &nbsp; &nbsp; 本着服务学生成长成才、服务企业自主创新、转型升级和汇聚人才的宗旨，联盟设立办公室，负责日常事务管理；设立活动部，负责联盟各项活动的计划、组织和实施，为联盟的和理事单位及其代表服务；科技部，负责日常的企业科技信息的收集、科技项目和合作项目的申报、学术支撑和科技活动的组织与落实等。</p><p> 联盟将常年提供和举行以下的平台和活动：</p><p style="margin-left: 31.5pt"> ?&nbsp;
