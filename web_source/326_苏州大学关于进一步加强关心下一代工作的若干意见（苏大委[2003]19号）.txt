COLOR: black; FONT-SIZE: 16px">是贯彻</SPAN><SPAN style="FONT-FAMILY: verdana; COLOR: black; FONT-SIZE: 16px">“</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">三个代表</SPAN><SPAN style="FONT-FAMILY: verdana; COLOR: black; FONT-SIZE: 16px">,</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">充分发挥老同志的优势</SPAN><SPAN style="FONT-FAMILY: verdana; COLOR: black; FONT-SIZE: 16px">,</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">)</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">及有关职能部门</SPAN><SPAN style="FONT-FAMILY: verdana; COLOR: black; FONT-SIZE: 16px">,</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">应把关心下一代工作列入议事日程</SPAN><SPAN style="FONT-FAMILY: verdana; COLOR: black; FONT-SIZE: 16px">,</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">切实加强领导</SPAN><SPAN style="FONT-FAMILY: verdana; COLOR: black; FONT-SIZE: 16px">,</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">重视和支持关心下一代工作</SPAN><SPAN style="FONT-FAMILY: verdana; COLOR: black; FONT-SIZE: 16px">,</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">定期听取汇报</SPAN><SPAN style="FONT-FAMILY: verdana; COLOR: black; FONT-SIZE: 16px">,</SPAN><SPAN style="FONT-FAMILY: 宋体; COLOR: black; FONT-SIZE: 16px">研究指导工作</SPAN><SPAN style="FONT-FAMILY: verdana;
