“微软学生软件资源点亮梦想计划”校内下载通道正式开通
发布者：发布时间：2010-12-01浏览次数：102
近日，我校与微软、中国教育网合作的“微软学生软件资源点亮梦想计划”校内下载通道正式开通了。我校同学可以登陆下载通道网站http://dreamspark.eol.cn，选择相应的软件，通过校内服务器的FTP进行下载。方便我校同学使用微软正版软件，便于课程学习以及创新实践。“微软学生软件资源--点亮梦想计划”提供的软件包括：1、Visual Studio 开发工具 和 SQL Server 数据库 将帮助学生进行面向对象的编程、数据库、软件开发、网络开发、嵌入式开发等课程的学习和课程实践。同时丰富的学习资源，将激发学生的科技创新热情，鼓励学生积极进行课内外创新实践。2、Expression 设计工具 将为学生在网页设计、平面设计以及动画/交互设计等方面带来更广阔的创作空间，帮助师生更自由的进行创意设计，更好的将创意变为现实。3、Windows Server 可用于操作系统原理相关课程的学习及实践，广大学生可以了解和学习构建基于Windows Server的高效、可靠和安全的IT基础架构。4、XNA 以及 Microsoft Robotics Studio是用于游戏开发和机器人开发的软件平台。具体详情请参考: http://dreamspark.eol.cn
